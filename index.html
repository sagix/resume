ARTE

au mois de fév. 2013 application ARTE

La mission consiste à réaliser une refonte de l'application ARTE Android afin de regrouper les fonctionnalités des applications tablettes et smartphones et d'y intégrer la nouvelle charte graphique.

L'application offre les fonctionnalités suivantes :

  • Lecture des programmes d'ARTE (replay) ;
  • Guide TV sur un mois ;
  • Programmes en LIVE ;
  • Accès aux différents services d’ARTE via une WebView.

Les objectifs étaient les suivants :

  • La fusion des applications tablettes et smartphones (mutualisation du code) ;
  • La migration des éléments TabActivity vers les Fragments offrant une meilleure intégration dans les versions récentes d'Android ;
  • L’intégration du nouveau design ;
  • Le téléchargement avec mise en cache d'une configuration distante ;
  • La synchronisation de la date et de la position géographique via un serveur afin de contrôler les droits de diffusion des émissions ;
  • La conversion d'ancien flux gérés par des WebServices à l'UTF-8 pour une meilleure gestion des caractères spéciaux ;
  • La gestion des flux vidéo de l'application pour permettre une reprise de la lecture en cours ;
  • L’ajout d'un WebView pour afficher les offres numériques avec gestion de la pagination ;
  • L’optimisation du code.

Compétences Techniques : Android (4.2) - XML - Json - Jackson - WebService – HTML
